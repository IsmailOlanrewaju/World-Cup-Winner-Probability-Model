# Data Description

This folder contains the datasets used in the World Cup Monte Carlo simulation model.

## team_data.csv

This file contains cleaned team-level performance statistics.

The original match-level data was aggregated into country-level averages.

Variables include:

- Country
- Average goals scored
- Average goals conceded
- Average shots
- Average shots on target
- Average xG
- Average xGA
- Average possession
- Average passes into final third
- Average opponent shots on target
- Average tackles won

Each row represents one national team.
